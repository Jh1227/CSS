문자열속성 선택자


선택자[속성 ~= 값] : 속성안의 값이 특정 값을 단어로 포함하는 태그를 선택

ex01 first

ex1-first

ex1 second

ex1 third


선택자[속성 |= 값] : 속성안의 값이 특정 값과 일치하거나, 지정 속성값 뒤에 하이픈("값-")으로 시작하는 태그를 선택

hello

hi

HI

하이


선택자[속성 ^= 값] : 속성안의 값이 특정 값으로 시작하는 태그를 선택

https://www.naver.com

http://www.naver.com


선택자[속성 $= 값] : 속성안의 값이 특정 값으로 끝나는 태그를 선택

test.html

test.css


선택자[속성 *= 값] : 속성안의 값에 특정 값을 포함하는 태그를 선택

first_test

first

test

span test
